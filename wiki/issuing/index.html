Issuing identifiers

Rule

Issue IDs; do not derive them from attributes.

For a new Entity

  1. Create data/E00000N.jsonld
  2. Create page/E00000N/index.html
  3. Create id/E00000N/index.html (resolver)
  4. Append an audit entry: audit/logs/00000N.json

Minimum JSON-LD fields

  • @context (ontology context)
  • @id (the canonical ID)
  • @type (Entity/Document/Event/Agent)
  • rdfs:label (human name)
  • registry:issuedAt (timestamp)
  • registry:status (active/deprecated/historical)
